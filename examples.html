showfor examples

View the source to see how it works.

Basic example

In this basic example one select simply relies on another. The selects have both numeric and alpha values. No config options are provided. The javascript is added inline after the HTML.

Options Example

This example shows how to use options. All breeds are available until the parent is chosen. If the parent is reset, all breeds are available again.

Recursive Example

The plugin will force an automatic change to trigger a DOM Change event for the child select. This allows for recursive showfor instances. No config, inline js.
